# Project Summary

## Overview
This project analyzes student academic performance using Power BI visualizations.

## Key Features
- Subject-wise average marks analysis
- Semester-wise performance comparison
- Department-wise attendance comparison
- Scatter plot showing relationship between attendance and marks
- Interactive department filter (slicer)

## Tools Used
- Power BI for data visualization
- GitHub for version control and project documentation
- CSV dataset for storing student data

## Outcome
The dashboard allows quick exploration of student performance trends and helps identify relationships between attendance and academic performance.
